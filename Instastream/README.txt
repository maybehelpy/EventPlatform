InstaStream v1.0
EventPlatform, maybehelpy
Пример работы - positivemexico

Возможности:
- Постраничный вывод фотографий из Instagram по заданному хэш-тегу для всех разрешений экранов 
- Включение режима фотографирования непосредственно приложением Instagram (для быстрой публикации фото)
- Вывод общего количества фото и лайков

Как добавить в проект:
1. Копируем файлы в стандартные для них места в проекте (html -> pages/ru/, css -> pages/, img -> images)
2. Далее как удобнее - либо из instastream.html копируем нужные части кода в рабочую страницу, либо дополняем её нужными частями из страницы проекта (в html добавлены только самые основные теги)
3. В файле instastream.html находим переменную HASHTAG, переименовываем в ПИНПРОЕКТА.
4. Создаём файл ПИНПРОЕКТА.json с пустым массивом (в тексте файла только "[]") на сервере ***.**.**.78 , путь C:\project\advisor\site\instagram
5. Профит, должно завестись
